# 🛠️ My OpenUSD Journey: From Colab to Local Dev

Welcome to my USD development repository! This project tracks my transition from cloud-based learning to a full professional local setup on Windows.

## 🚀 The Setup
I have successfully configured a local environment to author and visualize OpenUSD:
- **Language:** Python 3.11
- **Core Library:** `usd-core` (Pixar/OpenUSD)
- **Editor:** VS Code with USD Language Support
- **Engine:** NVIDIA Omniverse Kit SDK (Kit App Template)
- **CLI:** Git Bash for Windows

## 📂 Project Structure
- `/learning`: Experimental scripts and basic USD prim creation.
- `/assets`: Reusable USD components.
- `/scenes`: Complex staged compositions.
- `/my_usd_viewer`: A custom-built viewer using the NVIDIA Kit App Template.

## ✅ Milestones Reached
- [x] Installed local USD Python bindings.
- [x] Scripted a `my_first_scene.usd` containing a sphere and cube.
- [x] Built a standalone USD Viewer via NVIDIA Kit SDK.
